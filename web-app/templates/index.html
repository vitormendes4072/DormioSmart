Monitoramento de sono
não invasivo

Um dispositivo embarcado de baixo consumo, instalado dentro do travesseiro, que registra eventos de movimento durante o repouso — sem contato com o corpo.

O que o SmartDormio faz

Escopo declarado — sem promessas além do que o sistema entrega de verdade.

✅

O que registra

  • • Eventos de movimento durante o repouso
  • • Intensidade e horário de cada evento
  • • Períodos de ausência de movimento (repouso)
  • • Temperatura do sensor (chip MPU6050)
  • • Histórico com visualização em série temporal
⚠️

O que não faz

  • • Estadiamento de sono (REM, sono profundo, sono leve)
  • • Diagnóstico clínico ou médico
  • • Substituir polissonografia
  • • Medir temperatura do ambiente do quarto

Como funciona

Fluxo completo do sensor ao painel de visualização.

🛌

1. Sensor no travesseiro

ESP32 + MPU6050 embarcados no travesseiro captam aceleração e movimento sem contato com o corpo. O microcontrolador dorme em Deep Sleep e acorda apenas ao detectar movimento.

📡

2. Envio via Wi-Fi

A cada evento, o dispositivo transmite um pacote JSON via HTTPS para a API Flask hospedada na Vercel, que valida e persiste os dados no banco de dados Supabase.

📊

3. Visualização

O dashboard exibe o histórico de movimentação em série temporal, com os eventos, intensidades e status do sensor em tempo real.

MPU6050 ──I2C──› ESP32 ──HTTPS──› API Flask (Vercel) ──› Supabase ──› Dashboard
